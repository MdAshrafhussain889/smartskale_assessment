# Start the Application

## Step 1 — Start PostgreSQL & Redis

Open CMD:

```cmd
cd C:\Users\ASHRAF\Music\SmartSkale\SmartSkale\hiremind_backend
```

Run:

```cmd
docker compose up -d db redis
```

Verify:

```cmd
docker ps
```

Expected:

```text
hiremind_backend-db-1
hiremind_backend-redis-1
```

---

## Step 2 — Start Backend API

Open a second CMD window:

```cmd
cd C:\Users\ASHRAF\Music\SmartSkale\SmartSkale\hiremind_backend
```

Activate environment:

```cmd
venv\Scripts\activate
```

Start FastAPI:

```cmd
uvicorn app.main:app --reload --port 8000
```

Expected:

```text
Uvicorn running on http://127.0.0.1:8000
Application startup complete.
```

Verify:

```cmd
curl http://127.0.0.1:8000/health
```

Expected:

```json
{"status":"ok"}
```

Swagger API Documentation:

```text
http://127.0.0.1:8000/docs
```

---

# Start Judge0 (Coding Assessment Engine)

Navigate to backend:

```cmd
cd C:\Users\ASHRAF\Music\SmartSkale\SmartSkale\hiremind_backend
```

Start Judge0:

```cmd
docker compose -f docker-compose.judge0.yml up -d
```

Verify:

```cmd
docker ps
```

Expected:

```text
hiremind-judge0-server-1
hiremind-judge0-worker-1
hiremind-judge0-db-1
hiremind-judge0-redis-1
```

Stop Judge0:

```cmd
docker compose -f docker-compose.judge0.yml down
```
